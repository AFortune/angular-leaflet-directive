Simple map example

This is the simpliest way to embed a Leaflet map on our application. We only have to add angular-leaflet-map to our AngularJS application dependencies, and use our custom tag <leaflet> on the HTML code. As easy as pie.

  • HTML code
  • Javascript
var app = angular.module("basicMapApp", ["leaflet-directive"]); app.controller("BasicMapController", [ "$scope", function($scope) { // Nothing here! }]);

You can see a standalone example of this code here.

Customize markers example

You can interact with a leaflet marker with changed parameters on this code.

  • HTML code
  • Javascript Angular controller
var app = angular.module("mainPage", ['leaflet-directive']); app.controller("DraggingMarkersController", [ '$scope', function($scope) { angular.extend($scope, { center: { lat: 40.095, lng: -3.823, zoom: 6 }, markers: { Madrid: { lat: 40.095, lng: -3.823, message: "This is Madrid. Drag me to another position", focus: true, draggable: true }, Barcelona: { lat: 41.38, lng: 2.18, message: "This is Barcelona. You can't drag me", focus: false, draggable: false } } }); }]);

See how the form inputs change with the map interaction and viceversa. You can also see the code demonstrating this feature in the markers example demo.
