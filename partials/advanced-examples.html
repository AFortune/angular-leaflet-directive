Simple map example

This is the simpliest way to embed a Leaflet map on our application. We only have to add angular-leaflet-map to our AngularJS application dependencies, and use our custom tag <leaflet> on the HTML code. As easy as pie.

  • HTML code
  • Javascript
var app = angular.module("basicMapApp", ["leaflet-directive"]); app.controller("BasicMapController", [ "$scope", function($scope) { // Nothing here! }]);

Center example

Let's position our map. We can center and zoom an specific position of the map via a bi-directional property called center, which has 3 properties too: lat, lng and zoom. This properties could be changed on the fly, updating our map position, and viceversa.

  • HTML code
  • Javascript
app.controller("CenterController", [ '$scope', function($scope) { angular.extend($scope, { center: { lat: 40.095, lng: -3.823, zoom: 4 } }); }]);

You can see a standalone example of this code here.

Custom parameters example

Now that we know how to center the map to a selected position, we would like to change some default features or visualizations of the default map. We can use the defaults property to change the default behaviour of the map. We can override these properties via the defaults attribute. The options are pretty self-explanatory:

  • Default options
{ maxZoom: 14, minZoom: 1, doubleClickZoom: true, scrollWheelZoom: true, tileLayer: 'http://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', tileLayerOptions: { attribution: '© OpenStreetMap contributors' }, icon: { url: 'http://cdn.leafletjs.com/leaflet-0.5.1/images/marker-icon.png', retinaUrl: 'http://cdn.leafletjs.com/leaflet-0.5.1/images/marker-icon@2x.png', size: [25, 41], anchor: [12, 40], popup: [0, -40], shadow: { url: 'http://cdn.leafletjs.com/leaflet-0.5.1/images/marker-shadow.png', retinaUrl: 'http://cdn.leafletjs.com/leaflet-0.5.1/images/marker-shadow.png', size: [41, 41], anchor: [12, 40] } }, path: { weight: 10, opacity: 1, color: '#0000ff' }, center: { lat: 0, lng: 0, zoom: 10 } }

We can override some default options to obtain a map like the upper one you see on this example. Let's see the necessary code:

  • HTML code
  • Javascript
app.controller("CustomParametersController", [ '$scope', function($scope) { angular.extend($scope, { london: { lat: 51.505, lng: -0.09, zoom: 8 }, defaults: { tileLayer: "http://{s}.tile.opencyclemap.org/cycle/{z}/{x}/{y}.png", tileLayerOptions: { opacity: 0.9, detectRetina: true, reuseTiles: true, }, scrollWheelZoom: false } }); }]);

Dragging marker example

You can interact with a leaflet marker with this code on your HTML:

  • HTML code
  • Javascript Angular controller
var app = angular.module("mainPage", ['leaflet-directive']); app.controller("DraggingMarkerController", [ '$scope', function($scope) { angular.extend($scope, { center: { lat: 40.095, lng: -3.823, zoom: 4 }, markers: { Madrid: { lat: 40.095, lng: -3.823, message: "Drag me to another position", focus: true, draggable: true } } }); }]);

See how the form inputs change with the map interaction and viceversa. You can also see the code demonstrating this feature in the markers example demo.
