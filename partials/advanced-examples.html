Dragging marker example

You can interact with a leaflet marker with this code on your HTML:

<leaflet markers="markers" center="center"></leaflet>

And this code on your controller:

angular.extend($scope, {
    center: {
        lat: 40.095,
        lng: -3.823,
        zoom: 4
    },
    markers: {
        Madrid: {
            lat: 40.095,
            lng: -3.823,
            message: "Drag me to another position",
            focus: true,
            draggable: true
        }
    }
});

See how the form inputs change with the map interaction and viceversa. You can also see the code demonstrating this feature in the markers example demo.

Legend Demo

Maybe you're interested on a map including a legend with a list of elements identified on the map by color. This is accomplished with this directive:

<leaflet legend="legend" center="london"></leaflet>

We would need to add a property legend to the scope similar to this one:

$scope.legend = {
   position: 'bottomleft',
   colors: [ '#ff0000',
             '#28c9ff',
             '#0000ff',
             '#ecf386' ],
   labels: [ 'National Cycle Route',
             'Regional Cycle Route',
             'Local Cycle Network',
             'Cycleway' ]
};

And we would need to add a center attribute too:

$scope.london = {
    lat: 51.505,
    lng: -0.09,
    zoom: 12
};

Finally, we need some styles to decorate our leaflet legend:

.info {
    padding: 6px 8px;
    font: 14px/16px Arial, Helvetica, sans-serif;
    background: white;
    background: rgba(255,255,255,0.8);
    box-shadow: 0 0 15px rgba(0,0,0,0.2);
    border-radius: 5px;
}
.info h4 {
    margin: 0 0 5px;
    color: #777;
}
.legend {
    line-height: 18px;
    color: #555;
}
.legend i {
    width: 18px;
    height: 18px;
    float: left;
    margin-right: 8px;
    opacity: 0.7;
}

This example is based on this LeafletJS tutorial: http://leafletjs.com/examples/choropleth.html.
