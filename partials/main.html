Introduction

This is a very simple AngularJS directive developed to embed and interact with a map managed by the leaflet map library. Its not intended as a complete or full-of-features library, although its a simple and easy to use directive, which has been fun to develop and is a good starting point to learn to code a directive into the great AngularJS library.

The code of the library and this page has been inspired on the Angular Google Maps directive developed by nlaplante.

How to use it

Before using angular-leaflet-directive, you must include some javascript files to your main HTML: the main AngularJS library, the Leaflet library and the angular-leaflet-directive.js script. Something like this:

<script src="/path/to/leaflet.js"></script> <script src="/path/to/angular.min.js"></script> <script src="/path/to/angular-leaflet-directive.js"></script>

Also, you must include the CSS files of leaflet:

<link rel="stylesheet" href="/path/to//leaflet.css" />

And that's all. Now you can use the directive with the custom tag <leaflet> and some attributes, let's see them on the examples.
