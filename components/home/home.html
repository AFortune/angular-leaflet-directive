This directive allows you to embed a map on your AngularJS application and interact bidirectionally with it via the AngularJS scope and the leaflet map library API. Easy to use and extend, it's being evolved and improved constantly by a bunch of community contributors.

Let's see an step-by-step example of how to use it:

1. Loading the libraries

Before using angular-leaflet-directive, you must include these javascript libraries to your main HTML:

The AngularJS library.
The Leaflet library (javascript & CSS).
The angular-leaflet-directive.js.
