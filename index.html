angular-leaflet-directive
Simple to use AngularJS directive developed to ease embed and interact a map managed with leaflet map library.

This is a very simple AngularJS directive developed to embed and interact with a map managed by the leaflet map library. Its not intended as a complete or full-of-features library, although its a simple and easy to use directive, which has been fun to develop and is a good starting point to learn to code a directive into the great AngularJS library.
