Leaflet directive for Angular.js

Simple to use Angular.js directive to work with leaflet library and Open Street Maps inside your applications.

Code on Github Download

This is a very simple AngularJS directive developed to embed and interact with an OSM map managed by the great leaflet library. Its not intended as a complete a full-of-features library, although its a simple and working directive which has been fun to develop and has served me as the starting point to learn to develop a directive in AngularJS.

The code is mainly based on the Angular Google Maps library, and most of the features are based on the same features developed on that library.

Usage

Before using angular-leaflet-directive, you must include some files to your main HTML: the main Angular.js library, the Leaflet library and the angular-leaflet-directive.js script. Something like this:

<script src="/path/to//leaflet.js"></script>
<script src="/path/to/angular.min.js"></script>
<script src="/path/to/angular-leaflet-directive.js"<>/script>
<script src="/path/to/your-angular-controller.js"<>/script<

Optionally you can include the CSS files of leaflet and angular-leaflet-directive.

Now, you can use the directive with the custom tag <leaflet> and some attributes, let's see them on the examples.

Simple Demo

You can see the map embeded on the right-side of this page. This is accomplished width this directive: <leaflet></leaflet>


More complex Demo

You can interact with a leaflet marker with this code on your HTML:

<leaflet marker="marker" center="center" zoom="zoom" draggable="true"></leaflet>

And this code on your controller:

angular.extend($scope, {
    center: { lat: 40.094882122321145, lng: -3.8232421874999996 },
    marker: { lat: 40.094882122321145, lng: -3.8232421874999996 },
    message: "Drag me to your node position",
    zoom: 4
});

See how the form inputs change with the map interaction and viceversa.
